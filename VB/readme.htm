How to customize PDF Viewer Toolbar Items

The following example shows how to customize PDF Viewer Ribbon. Starting with version 14.1.5, this can be done using actions (https://documentation.devexpress.com/#WPF/CustomDocument7045). In previous versions, it is necessary to override the default PdfViewerControl's RibbonTemplate and add the required bar items to the newly defined RibbonControl.

You can find sample updates and versions for different programming languages here:
http://www.devexpress.com/example=E5091.
